# Dart 项目实战路线与练习建议

## 1. 控制台工具类项目（纯 Dart）

### 项目目标

掌握 Dart 基础语法、流程控制、函数、类、文件操作、异常处理、命令行交互等。

### 推荐项目与功能拆解

#### 1.1 记账本

- 命令行输入/输出，支持添加、删除、查询、修改账目
- 数据持久化到本地 JSON 或 CSV 文件
- 支持按日期/类型/金额筛选与统计
- 导出账单报表

#### 1.2 简易日历/倒计时工具

- 显示当前月日历、支持添加提醒
- 倒计时功能，支持自定义目标日期

#### 1.3 批量文件重命名脚本

- 读取指定目录下所有文件，批量重命名（如加前缀、后缀、序号）
- 支持撤销操作

### 技术要点

- Dart IO 库（dart:io）文件读写、目录操作
- 字符串处理、正则表达式
- 面向对象设计
- 错误与异常处理

### 进阶建议

- 增加单元测试
- 支持命令行参数解析

## 2. 网络爬虫/数据处理项目

### 项目目标

练习 Dart 的 http 请求、JSON 解析、异步编程、异常处理、数据清洗与转换。

### 推荐项目与功能拆解

#### 2.1 天气/新闻/汇率爬虫

- 调用开放 API 获取数据（如天气、新闻、汇率）
- 支持定时抓取、数据去重
- 保存为本地 JSON 文件
- 简单命令行展示

#### 2.2 本地数据处理工具

- 读取本地 CSV/JSON 文件
- 数据筛选、分组、统计（如求和、平均值、最大最小值）
- 支持导出为新文件

### 技术要点

- http 包网络请求
- dart:convert 进行 JSON 解析与序列化
- 异步编程（Future、async/await）
- 文件读写与异常处理

### 进阶建议

- 增加数据可视化（如生成简单文本图表）
- 支持多线程（isolate）加速大数据处理

## 3. 服务端 API 小项目

### 项目目标

体验 Dart 在服务端的用法，理解路由、请求处理、响应、依赖管理、RESTful 设计。

### 推荐项目与功能拆解

#### 3.1 待办事项 RESTful API

- 使用 shelf 框架搭建 HTTP 服务
- 实现增删查改（CRUD）接口
- 数据存储可用内存、JSON 文件或 SQLite
- 支持跨域（CORS）、简单鉴权

#### 3.2 简易博客 API

- 文章增删查改、评论管理
- 支持分页、搜索

### 技术要点

- shelf、shelf_router 等服务端框架
- 路由与中间件
- JSON 解析与响应
- 异步与异常处理

### 进阶建议

- 增加单元测试、接口文档（OpenAPI）
- Docker 化部署

## 4. Flutter 小应用（进阶）

### 项目目标

将 Dart 语言与 Flutter 结合，体验 UI 构建、状态管理、网络请求、数据持久化、动画等。

### 推荐项目与功能拆解

#### 4.1 记事本/待办事项 App

- 列表展示、添加、编辑、删除事项
- 本地存储（shared_preferences、sqflite、hive）
- 支持搜索、分类、标记完成
- UI 主题切换

#### 4.2 天气查询 App

- 输入城市，调用天气 API 获取并展示天气信息
- 支持多城市管理、天气图标、未来预报
- 网络异常处理、加载动画

#### 4.3 简易博客/留言板

- 文章/留言增删查改，支持评论
- 网络请求与数据同步
- 列表分页、下拉刷新、上拉加载

### 技术要点

- Flutter UI 构建与布局
- 状态管理（setState、Provider、Bloc 等）
- 网络请求与 JSON 解析
- 本地存储与缓存
- 动画与交互

### 进阶建议

- 增加单元测试、UI 测试
- 支持多端适配（Web、桌面）

---

### 项目练习建议

- 先用 Dart 纯语法实现核心逻辑，再逐步引入第三方库和框架
- 代码多写注释，遇到问题多查官方文档
- 项目完成后，尝试重构、加测试、优化性能
- 多与社区交流，参考优秀开源项目
